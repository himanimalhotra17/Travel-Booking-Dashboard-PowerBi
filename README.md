# Travel Booking Dashboard Analysis (Power BI)

## Project Overview

This project presents an interactive Power BI dashboard analyzing travel booking data to uncover insights into revenue trends, customer behavior, and booking performance.

The dashboard enables stakeholders to monitor KPIs, identify growth opportunities, and optimize business strategies using data-driven insights.

Kaggle Notebook:
https://www.kaggle.com/code/himanimalhotra/power-bi-travel-booking-dashboard-project

---

## Business Objective

The objective of this project is to analyze travel booking performance and answer key business questions:

- What are the revenue trends over time?
- Which categories generate the highest revenue?
- What are the booking patterns across customer segments?
- Where are potential growth opportunities?

---

## Tools Used

- Power BI (Dashboard & Visualization)
- Excel / CSV (Data Preparation)
- Kaggle (Project Hosting)

---

## Dataset Overview
The dataset included transactional booking-level data with the following key fields:
• Booking ID
• Booking Date
• Category
• Channel
• City
• Booking Status
• Base Price
• Final Price
• Discount Percentage
• Net Revenue
• Refund Amount
The dataset enabled multi-dimensional analysis across time, geography, category, and acquisition channel.
